📖 PRD : Programmation Web Dynamique (PHP)

1. Concepts Clés & Architecture

  • Web Dynamique : Contrairement au statique, le serveur génère le code HTML à la volée pour chaque client.
  • Architecture : Le Front-End s'exécute dans le navigateur (HTML/CSS/JS), tandis que le Back-End s'exécute sur le serveur (PHP/Bases de données).
  • API REST : Pont de communication utilisant les méthodes HTTP (GET, POST, PUT, DELETE) pour échanger des données au format JSON.

2. Fondamentaux du PHP

PHP est un langage de script interprété côté serveur[cite: 1]. Toute instruction se termine par un point-virgule (;)[cite: 1].

A. Intégration dans le HTML

<?php
echo "Je suis un étudiant.";
?>
Explication : Le code est encadré par les balises <?php et ?>[cite: 1]. L'instruction echo est utilisée pour envoyer du texte vers le navigateur afin qu'il soit affiché[cite: 1].

B. Guillemets Doubles vs Simples

<?php 
$nom = "Toto";
echo "Hello $nom"; // Affiche : Hello Toto
echo 'Hello $nom'; // Affiche : Hello $nom
?>
Explication : Avec les guillemets doubles, PHP interprète les variables à l'intérieur[cite: 1]. Avec les guillemets simples, le texte est traité de manière brute et la variable n'est pas remplacée par sa valeur[cite: 1].

3. Variables, Constantes et Types

  • Variables : Précédées du signe $. Le typage est faible (automatique)[cite: 1].
  • Constantes : Définies avec define("NOM", valeur). Elles n'ont pas de signe $ et sont sensibles à la casse[cite: 1].
  • Types de base : Entiers (int), Flottants (float), Booléens (bool), Chaînes (string), Tableaux (array) et Objets[cite: 1].

4. Structures de Contrôle

La condition "Switch"

<?php
$operation = "+";
$a = 5; $b = 3;
switch ($operation) {
    case "+":
        echo $a + $b; break;
    case "-":
        echo $a - $b; break;
    default:
        echo "Opération inconnue.";
}
?>
Explication : Le switch compare une variable à plusieurs cas (case). Le mot-clé break est indispensable pour arrêter l'exécution une fois qu'un cas correspond, sinon PHP continue de lire les suivants[cite: 1].

5. Manipulation de Chaînes (Le Top 8)

  • strlen($ch) : Longueur de la chaîne[cite: 1].
  • strtolower() / strtoupper() : Changement de casse[cite: 1].
  • trim() : Supprime les espaces inutiles[cite: 1].
  • strpos($ch, $recherche) : Position d'un mot[cite: 1].
  • substr($ch, debut, taille) : Extraction d'une partie[cite: 1].
  • str_replace(ancien, nouveau, $ch) : Remplacement de texte[cite: 1].
  • explode(separateur, $ch) : Coupe une chaîne en tableau[cite: 1].
  • implode(separateur, $tab) : Joint un tableau en chaîne[cite: 1].

6. Comparaison Stricte

<?php
$a = 5;    // entier
$b = "5";  // chaîne
if ($a == $b)  { /* VRAI : même valeur */ }
if ($a === $b) { /* FAUX : types différents */ }
?>
Explication : L'opérateur == vérifie uniquement la valeur[cite: 1]. L'opérateur === (comparaison stricte) vérifie la valeur ET le type de donnée[cite: 1].
